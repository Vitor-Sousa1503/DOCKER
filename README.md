# DOCKER
**10 comandos docker linux**

## 1° docker --versions
**Mostra a versão do docker instalado**

## 2° docker ps
**Monstra os containers que estão rodando**

## 3° docker ps -a
**Mostra todos os containers, inclusives os que estão parados**

## 4° docker images
**Lista todas as imagens docker instaladas**

## 5° docker run hello-world
**Executa um container de teste**

## 6° docker pull hello-world
**Baixa uma imagem do docker hub**

## 7° docket stop id-do-container
**Para um container em execução**

## 8° docker rm id-do-container
**Remove um container**

## 9° docker start id-do-container
**Inicia um conteiner**

## 10° docker rmi nome-da-imagem
**Remove uma imagem docker**
